# SimuladorDeCajeroAutomaticoBasico
Proyecto básico para en consola que simule operaciones básicas de un cajero automático.

## Prerrequisitos
- Instalar python3 en el sistema operativo.
- Instalar Visual Studio Code.
- Instalar las extenciones oficiales de Python.
- Tener conocimientos de ciclos y condicionales.
- Tener conocimiento en el manejo de errores (Try - Except).
